# Bingda Robot Simulation Package

## Overview

This repository contains the simulation package and environment description for the **Bingda robot platform**, including robot model and simulation maps.

It is based on the original official package provided by Bingda Robotics:

https://gitee.com/bingda-robot/nanorobot_description

---

## Purpose of This Repository

This repository does not introduce new simulation models or algorithms.

It mainly includes:
- Parameter tuning for the robot simulation model
- Addition and extension of simulation environments and maps
- Adaptation for experimental scenarios in research work

---

## Modifications

Compared with the original version, this repository includes:
- Adjusted robot simulation parameters
- Added multiple simulation worlds/maps
- Optimized environment settings for reproducible experiments

---

## Original Source

All original simulation models and descriptions are provided by Bingda Robotics:
